        This website showcases our collection of authentic Ethiopian garments, offering customers a unique shopping experience.

    <h2>Technologies Used</h2>
    <ul>
        <li>HTML: Used for structuring the web pages.</li>
        <li>CSS: Used for styling the website and making it visually appealing.</li>
        <li>jQuery: Used to enhance user interactions and provide dynamic features.</li>
        <li>PHP: Used for server-side scripting and handling backend functionality.</li>
    </ul>

    <h2>Getting Started</h2>
    <p>
        To run the "Made in Ethiopia" website locally, follow these steps:
    </p>
    <ol>
        <li>Clone the repository: <code>git clone https://github.com/solo21-12/made-in-ethiopia.git</code></li>
        <li>Navigate to the project directory: <code>cd made-in-ethiopia</code></li>
        <li>Start a local development server.</li>
        <li>Access the website in your browser at: <code>http://localhost:8000</code></li>
    </ol>

    <h2>Features</h2>
    <ul>
        <li>Browse and view a wide variety of traditional Ethiopian clothing.</li>
